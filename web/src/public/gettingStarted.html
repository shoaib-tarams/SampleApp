Docker

Prerequisites

Docker installed on your machine


  1. Pull the sample application source code from GitHub and use as your application in the Getting Started Wizard
    git clone https://github.com/Appdynamics/SampleApp.git
  2. cd SampleApp
  3. Download the Agent
    curl http:download.appdynamics.com -- output AppServerAgent.zip
  4. Build and run the Docker container
    docker-compose up
